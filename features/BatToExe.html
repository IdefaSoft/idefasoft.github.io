Fonctionnalités de Bat To Exe Converter

Voici la page dans laquelle les fonctionnalités principales de Bat To Exe Converter sont listées.

Un logiciel permettant de convertir du batch (des fichiers .bat) en exe.
Voici ses principales caractéristiques :

  1. Les fichiers compilés sont très légers.
  2. Les fichiers peuvent être compilés pour les architectures 32 bits ou 64 bits (au choix)
  3. Il existe deux thèmes : le mode sombre et le mode clair.
  4. Une icône peut être rajoutée aux fichiers compilés.
  5. Il peut compiler aussi bien des fichiers que du texte saisi dans la zone de texte.
