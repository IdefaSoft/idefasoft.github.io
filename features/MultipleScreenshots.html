Fonctionnalités de MultipleScreenshots

Voici la page dans laquelle les fonctionnalités principales de MultipleScreenshots sont listées.

MultipleScreenshots est un petit logiciel léger permettant de réaliser un certain nombre de captures d'écran à intervalle régulier.
Voici ses principales caractéristiques :

  1. Il permet de prendre le nombre de captures d'écran (aucune limite) au même intervalle de temps.
  2. Il dispose de 8 formats d'images de sorties.
  3. L'intervalle entre chaque capture est presque entièrement personnalisable.
  4. L'utilisateur peut choisir le dossier de sortie.
  5. ScreenSoft, un logiciel permettant de réaliser une capture d'écran prévisualisable et enregisrable, est inclus dedans.
  6. Le nom d'enregistrement des fichiers est entièrement personnalisable, l'utilisateur peut, en plus du nom du fichier qu'il veut, mettre des variables comme l'année, le jour, l'heure d'enregistrement de la capture, le nom de la machine ou encore le nom d'utilisateur.
    Voici la liste de toutes ces variables :
    1. $A : l'année en cours
    2. $M : le mois en cours
    3. $J : le jour en cours
    4. $H : l'heure en cours
    5. $m : la minute en cours
    6. $S : la seconde en cours
    7. $User : le nom de l'utilisateur de l'ordinateur
    8. $Machine : le nom de la machine

    Ainsi, si l'utilisateur rentre "$A-$M-$J_$H-$m-$s", l'image numéro 96 sera par exemple nommée "2022-01-25_16-54-17_96.png".
