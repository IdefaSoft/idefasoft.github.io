IdefaSoft est une marque de logiciels utiles et gratuits.

Fonctionnalités des logiciels

Voici la page dans laquelle les fonctionnalité principales des logiciels sont listées.

1 MultipleScreenshots

MultipleScreenshots est un petit logiciel léger permettant de réaliser un certain nombre de captures d'écran à intervalle régulier.
Voici ses principales caractéristiques :

  1. Il permet de prendre le nombre de captures d'écran (aucune limite) au même intervalle de temps.
  2. Il dispose de 8 formats d'images de sorties.
  3. L'intervalle entre chaque capture est presque entièrement personnalisable.
  4. L'utilisateur peut choisir le dossier de sortie.
  5. ScreenSoft, un logiciel permettant de réaliser une capture d'écran prévisualisable et enregisrable, est inclus dedans.
  6. Le nom d'enregistrement des fichiers est entièrement personnalisable, l'utilisateur peut, en plus du nom du fichier qu'il veut, mettre des variables comme l'année, le jour, l'heure d'enregistrement de la capture, le nom de la machine ou encore le nom d'utilisateur.
    Voici la liste de toutes ces variables :
    1. $A : l'année en cours
    2. $M : le mois en cours
    3. $J : le jour en cours
    4. $H : l'heure en cours
    5. $m : la minute en cours
    6. $S : la seconde en cours
    7. $User : le nom d l'utilisateur de l'ordinateur
    8. $Machine : le nom de la machine

    Ainsi, si l'utilisateur rentre "$A-$M-$J_$H-$m-$s", l'image numéro 96 sera par exemple nommée "2022-01-25_16-54-17_96.png".

2 VigiFolder

VigiFolder est un logiciel léger permettant de surveiller la plupart des changements (tous ?) effectués dans un dossier (suppression, renommage, modification, création).
Voici ses principales caractéristiques :

  1. Il permet de surveiller les changements effectués même dans les sous dossiers du dossier sélectionné.
  2. Il permet d'exporter les données dans un fichier rtf.
  3. Si les données sont trop nombreuses, un bouton permet de toutes les supprimer.

3 IdefaNote

Un logiciel alternatif au bloc note de windows, avec de multiples options
Voici ses principales caractéristiques :

  1. La police, la couleur, la taille et l'alignement du texte sont modifiables
  2. Les notes sont enregistrées au format rtf (bien sûr, il peut ouvrir une note enregistrée).
  3. Il existe deux thèmes : le mode sombre et le mode clair. (les préférences sur le thème sont enregistrées)
  4. Les notes peuvent être ouverte en lecture seul.
  5. Lorsque l'application est masqué dans la barre des tâches, de multiples options avec un clic droit sont disponibles.
  6. Des raccourcis comme Ctrl + W pour fermer ou Ctrl + S pour enregistrer sont présents.

4 TextSender

Un logiciel permettant de simuler l'appui de touches du clavier, avec de nombreuses options disponibles.
Voici ses principales caractéristiques :

  1. Jusqu'à 3 textes différents peuvent etre envoyés.
  2. Le texte peut être envoyé avec un intervalle variant de 8 ms à plusieurs jours.
  3. Il existe deux thèmes : le mode sombre et le mode clair.
  4. On peut chosir le délai avant que le texte ne commence à être écrit.
  5. L'appui de la touche entrée après avoir écrit chaque textes est facultative.

5 Bat To Exe Converter

Un logiciel permettant de convertir du batch (des fichiers .bat) en exe.
Voici ses principales caractéristiques :

  1. Les fichiers compilés sont très légers.
  2. Les fichiers peuvent être compilés pour les architectures 32 bits ou 64 bits (au choix)
  3. Il existe deux thèmes : le mode sombre et le mode clair.
  4. Une icône peut être rajoutée aux fichiers compilés.
  5. Il peut compiler aussi bien des fichiers que du texte saisi dans la zone de texte.

6 Trieur de fichiers

Trieur de fichiers est un logiciel triant les fichiers d'un dossier par extension en créant d'autres dossiers dans lequels ils seront déplacés.
Voici ses principales caractéristiques :

  1. Plus de 70 extensions de fichiers sont supportées, les autres extensions sont déplacées dans un dossier "Autres".
  2. Le logiciel est assez rapide.
  3. Bien qu'il n'ait pas d'interface graphique, il est très simple d'utilisation.

7 Pays et capitales

Pays et capitales est un logiciel permettant d'interroger l'utilisateur sur les pays et les capitales, avec différents modes possibles.
Voici ses principales caractéristiques :

  1. Il permet d'interroger l'utilisateur sur autant de pays ou capitales qu'il le souhaite.
  2. Il permet de visualiser un tableau avec la liste des pays avec leurs capitales.
  3. Bien qu'il n'ait pas d'interface graphique, il est très simple d'utilisation.

8 Cryptographie

Cryptographie est un logiciel qui crypte du texte en utilisant des codages très simple, il peut aussi crypter des fichiers textes.
Voici ses principales caractéristiques :

  1. Il permet de crypter du texte écrit directement dans le logiciel, ainsi que du texte provenant de fichiers textes.
  2. Il peut aussi décrypter le texte crypté.
  3. Bien qu'il n'ait pas d'interface graphique, il est très simple d'utilisation.
