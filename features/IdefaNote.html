Fonctionnalités de IdefaNote

Voici la page dans laquelle les fonctionnalités principales de IdefaNote sont listées.

Un logiciel alternatif au bloc note de windows, avec de multiples options
Voici ses principales caractéristiques :

  1. La police, la couleur, la taille et l'alignement du texte sont modifiables
  2. Les notes sont enregistrées au format rtf (bien sûr, il peut ouvrir une note enregistrée).
  3. Il existe deux thèmes : le mode sombre et le mode clair. (les préférences sur le thème sont enregistrées)
  4. Les notes peuvent être ouverte en lecture seul.
  5. Lorsque l'application est masqué dans la barre des tâches, de multiples options avec un clic droit sont disponibles.
  6. Des raccourcis comme Ctrl + W pour fermer ou Ctrl + S pour enregistrer sont présents.
