Fonctionnalités de VigiFolder

Voici la page dans laquelle les fonctionnalités principales de VigiFolder sont listées.

VigiFolder est un logiciel léger permettant de surveiller la plupart des changements (tous ?) effectués dans un dossier (suppression, renommage, modification, création).
Voici ses principales caractéristiques :

  1. Il permet de surveiller les changements effectués même dans les sous dossiers du dossier sélectionné.
  2. Il permet d'exporter les données dans un fichier rtf.
  3. Si les données sont trop nombreuses, un bouton permet de toutes les supprimer.
