Fonctionnalités de TextSender

Voici la page dans laquelle les fonctionnalités principales de TextSender sont listées.

TextSender est un logiciel permettant de simuler l'appui de touches du clavier, avec de nombreuses options disponibles.
Voici ses principales caractéristiques :

  1. Jusqu'à 3 textes différents peuvent être envoyés.
  2. Le texte peut être envoyé avec un intervalle variant de 8 ms à plusieurs jours.
  3. Il existe deux thèmes : le mode sombre et le mode clair.
  4. On peut choisir le délai avant que le texte ne commence à être écrit.
  5. L'appui de la touche entrée après avoir écrit chaque texte est facultative.
